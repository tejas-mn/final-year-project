Features

The project employs Deep Learning Algorithms to forecast crop diseases. It utilizes diverse deep learning libraries to construct a resilient and efficient neural network, enabling rapid and precise outcomes.

  • Mobile Responsive Website
  • Capture images in real time with camera on Mobile Devices
  • Auto deletion of images which saves server storage space
  • Gives the latest articles and news related to predicted disease
  • Displays fungicides for the predicted disease to buy
  • Shows the healthy part and diseased part with percentage of area infected
  • Google search results for the predicted disease
  • Gives the symptoms and ways to manage and control the disease

System Architecture

The System comprises of frontend built using bootstrap framework for fast development and its responsive features. Flask Web Framework is used for serving the tensorflow model in backend which takes the uploaded image as input and gives the response for the predicted disease. The CNN models are trained on dataset consisting around 3000 images collected from various datasets like PlantVillage, PLD, Mendely and also images from web.

Challenges Faced:

  • Image compression implemented using Javascript at Client Side.
  • Handling Image size for faster processing and reducing Time Complexity.
  • Automatic deletion of images using threading to reduce server storage space.
  • Need for various datasets for diverse image classification.

About

This project uses Convolutional Neural Networks and image processing to detect and diagnose diseases in potato leaves based on their leaf characteristics. Our system analyzes leaf images, extracting features and using them to classify diseases.
